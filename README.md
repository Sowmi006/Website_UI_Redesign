# Website_UI_Redesign

IRCTC Website UI Redesign: 

📌 Project Title

IRCTC Website UI Redesign (Task 2 – Thiranex Internship)

🎯 Objective

To redesign an existing IRCTC website homepage with a modern, clean, and user-friendly interface.

🛠 Tools Used

Canva (for UI redesign)
UI/UX principles

🔍 Problem Analysis (Old UI Issues)

Outdated design and color scheme
Poor spacing and alignment
Cluttered layout with too many elements
Confusing navigation structure
Low visual hierarchy and readability

🎨 Improvements in Redesigned UI

Modern blue & white color palette
Clean header with simple navigation
Hero section with clear booking message
Card-based layout for features
Improved search booking section
Better spacing and typography (Poppins font)
Responsive mobile-friendly layout

🧠 Outcome

The redesigned UI improves usability, clarity, and modern design standards, making ticket booking easier and more user-friendly.
